Запуск: 
```shell
yarn start:dev
````

Поднять окружение:
```shell
docker-compose up
````
env example: 
```shell
PORT=5000
POSTGRES_HOST=[docker ? postgres : localhost]
POSTGRES_USER=postgres
POSTGRES_DB=test
POSTGRES_PASSWORD=root
POSTGRES_PORT=5432
PRIVATE_KEY=SECRET
```

Документация: http://localhost:5000/api/docs
